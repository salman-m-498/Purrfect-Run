SKATEBOARDING CAT GAME
====================

A physics-based skateboarding game where you control a cat performing tricks and combos.

Controls
--------
- Mouse/Touch Movement: Swipe to control the skateboard
- Hold and Release: Jump and perform tricks
- Multiple Swipes: Perform multiple flips
- Swipe Direction:
  * Up/Down: Front flip / Back flip
  * Left/Right: Kick flip / Heel flip

Trick System
-----------
- Basic Tricks: Single flips in different directions
- Advanced Tricks: Multiple flips can be performed in the air
- Combo System: Chain tricks together within a time window for higher scores
- Landing: Must land properly to complete tricks and score points

Tips
----
1. Time your jumps carefully to get maximum height
2. Start with single flips before attempting multiple flips
3. Make sure to land flat for successful tricks
4. Chain tricks together for combo multipliers
5. Practice timing for smooth landing transitions
